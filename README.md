NSGA-III-based Multi-Criteria Optimization for Information Security Systems

This repository implements a multi-objective optimization framework for selecting information security systems in distributed university networks using NSGA-III with bilinear game-theoretic modeling.

## Features
- NSGA-III optimization
- Multi-criteria decision making
- Game-theoretic modeling
- Reproducible workflow
